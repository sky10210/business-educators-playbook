Whole-Class Engagement

Create visible opinions, purposeful movement, debate, discussion, and shared meaning-making when the entire room should react, compare, and respond.

Browse Strategies Use the Activity Finder
Best used when
  • The whole room should react to the same prompt
  • You want movement with an academic purpose
  • Students should hear competing viewpoints
  • A shared debrief is central to the lesson
